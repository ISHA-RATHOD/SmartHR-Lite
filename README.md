# SmartHR Lite (Under Development)

### Overview
SmartHR Lite is a modular HRMS component designed to streamline employee attendance tracking and leave management workflows. This project is currently in the development phase, focusing on implementing secure and automated HR workflows using ASP.NET Core MVC.

### Tech Stack
* **Framework:** ASP.NET Core MVC
* **Language:** C#
* **Database:** SQL Server (Entity Framework Core)
* **Frontend:** Bootstrap 5, JavaScript

### Key Features (Planned)
* **Leave Management:** Automated approval workflows for leave requests.
* **Attendance Tracking:** Real-time logging and reporting using LINQ.
* **Role-Based Access:** Secure admin and employee dashboards.

### Current Status
* [x] Database Schema Design (Normalized)

* [ ] Leave Approval Workflow Logic (In Progress)
